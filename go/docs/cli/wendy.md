## wendy

Wendy CLI - Edge Computing Development Tool

### Synopsis

Wendy is a CLI for developing and deploying edge computing applications to WendyOS devices.

### Options

```
      --device string   Target device hostname
  -h, --help            help for wendy
      --json            Output in JSON format
```
