## wendy audio listen

Stream raw audio from a device microphone

```
wendy audio listen [flags]
```

### Options

```
      --channels uint32      Number of audio channels (default 1)
  -h, --help                 help for listen
      --id uint32            Audio device ID
      --sample-rate uint32   Sample rate in Hz (default 16000)
```

### Options inherited from parent commands

```

      --json            Output in JSON format
```
