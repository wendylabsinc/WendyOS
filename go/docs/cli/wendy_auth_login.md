## wendy auth login

Log in to Wendy Cloud

### Synopsis

Opens a browser for authentication, receives a callback with an enrollment token, generates certificates, and saves them to config.

```
wendy auth login [flags]
```

### Options

```
      --cloud string        Cloud dashboard URL
      --cloud-grpc string   Cloud gRPC endpoint
  -h, --help                help for login
```
