## wendy device logs

Stream logs from containers on the device

```
wendy device logs [flags]
```

### Options

```
      --app string           Filter by application name
  -h, --help                 help for logs
      --level string         Minimum log level (trace, debug, info, warn, error, fatal)
      --min-severity int32   Minimum log severity number
      --service string       Filter by service name
```

### Options inherited from parent commands

```
      --device string   Target device hostname
      --json            Output in JSON format
```
