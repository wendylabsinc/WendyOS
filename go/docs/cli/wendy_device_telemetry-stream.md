## wendy device telemetry-stream

Stream telemetry data (logs, metrics, traces) as JSONL

```
wendy device telemetry-stream [flags]
```

### Options

```
      --app string       Filter by application name
  -h, --help             help for telemetry-stream
      --logs             Include logs
      --metrics          Include metrics
      --service string   Filter by service name
      --traces           Include traces
```

### Options inherited from parent commands

```
      --device string   Target device hostname
      --json            Output in JSON format
```
