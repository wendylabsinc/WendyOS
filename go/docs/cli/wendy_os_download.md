## wendy os download

Download a WendyOS image to the local cache

### Synopsis

Download (and extract) a WendyOS image for a supported device. The image is stored in the local cache for later use by `wendy os install`.

```
wendy os download [flags]
```

### Options

```
  -h, --help             help for download
      --overwrite        Overwrite cached image without prompting
      --version string   OS version to download (interactive if omitted)
```
